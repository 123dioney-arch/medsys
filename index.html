⚙️ Configurar servidor de actualizaciones

El sistema busca actualizaciones en la URL que configures. Puedes usar GitHub (gratis) o cualquier servidor web. Cada actualización conserva tus expedientes y datos.

📖 Cómo configurar GitHub (gratuito):
  1. Crea una cuenta en github.com (gratis)
  2. Crea un repositorio nuevo llamado medsys
  3. Sube el archivo version.json (ve el ejemplo abajo)
  4. Copia la URL "Raw" del archivo y pégala arriba
  5. Cuando quieras actualizar, sube el nuevo HTML y edita version.json
📄 Ejemplo de version.json
{
  "version": "10.1.0",
  "fecha": "2026-04-22",
  "url": "https://raw.githubusercontent.com/TU_USUARIO/medsys/main/MedSys.html",
  "notas": [
    "✅ 50 nuevos medicamentos agregados",
    "🐛 Corrección en generación de recetas",
    "⚡ Mejora de velocidad en búsqueda"
  ],
  "critica": false
}

Instalar MedSys como app
Funciona sin internet, abre desde el escritorio o pantalla de inicio, y se ve como una app nativa — sin barra del navegador.
Si el botón no responde, usa las instrucciones de tu dispositivo abajo.
🤖 Android (Chrome)
  1. Abre MedSys en Chrome
  2. Toca el menú ⋮ (tres puntos arriba)
  3. Selecciona "Instalar app" o "Agregar a pantalla de inicio"
  4. Confirma → aparece el ícono 🏥 en tu pantalla
💡 También aparece el botón 📲 Instalar app en la barra superior del sistema
🍎 iPhone / iPad (Safari)
  1. Abre MedSys en Safari (obligatorio, no Chrome)
  2. Toca el botón □↑ Compartir (abajo en iPhone, arriba en iPad)
  3. Desliza y toca "Agregar a pantalla de inicio"
  4. Confirma con Agregar
⚠️ En iPhone solo funciona desde Safari — Chrome en iOS no soporta instalación
🖥️ Windows / Mac (Chrome o Edge)
  1. Abre MedSys en Chrome o Edge
  2. En la barra de dirección busca el ícono ⊕ Instalar (a la derecha)
  3. Haz clic → Instalar
  4. Abre desde el menú Inicio o escritorio como cualquier programa
💡 En Edge: menú ⋯ → Aplicaciones → Instalar este sitio como aplicación
📁 Desde archivo local
Si usas MedSys como archivo .html (sin servidor), la instalación como PWA requiere un servidor web. Opciones sencillas:
  1. Sube MedSys.html a GitHub Pages (gratis)
  2. Comparte esa URL con todos los usuarios
  3. Cada uno instala desde su dispositivo con los pasos de arriba
💡 GitHub Pages es gratis — el mismo repositorio que usas para actualizaciones sirve para hospedar la app
🚀 Activar GitHub Pages — Hosting gratuito en 3 pasos
① Repositorio
En tu repositorio medsys en GitHub, ve a Settings → Pages
② Activar
En Source selecciona Deploy from a branch → main → / (root) y guarda
③ Renombrar
Renombra MedSys.html a index.html en GitHub. La URL será:
tuusuario.github.io/medsys
✅ Una vez en GitHub Pages, cualquier persona que abra tuusuario.github.io/medsys podrá instalar MedSys como app en su dispositivo. Los datos siguen guardándose localmente en cada dispositivo — no se suben a ningún servidor.
